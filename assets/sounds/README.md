# Sound Files for Gumball Game

Add audio files here for answer feedback. The game plays random sounds for correct and wrong answers.

## Required Files

### Correct Answer Sounds (Practice Mode Only - 3 files)
- `correct1.mp3` - "Well done!", "Good job!"
- `correct2.mp3` - Alternative celebratory sound
- `correct3.mp3` - Another celebratory sound

### Wrong Answer Sounds (Practice Mode Only - 3 files)
- `wrong1.mp3` - "Oops!", "Try again!", sad sound
- `wrong2.mp3` - Alternative failure sound
- `wrong3.mp3` - Another failure/buzzer sound

### Game End Sounds (Speed Race & Boss Battle - 2 files)
- `win.mp3` - Victory/celebration sound (plays when accuracy >= 75%)
- `lose.mp3` - Loss/sad sound (plays when accuracy < 75%)

**Note:** Practice mode plays sounds for each answer. Speed Race and Boss Battle only play sounds at the end.

## How to Get Audio

**Option 1: From Show Clips**
- Extract from Gumball episodes using video editing software
- Trim phrases to 2-3 seconds
- Convert to MP3 format

**Option 2: Free Sound Libraries**
- Freesound.org - search "well done", "oops", "wrong", "buzzer"
- Zapsplat.com - royalty-free effects
- Pixabay.com - free audio

**Option 3: Text-to-Speech**
- Use TTS for phrases like "Good job!", "Try again!"
- Add celebratory/sad sound effects

## File Format
- Format: MP3 (or WAV/OGG)
- Duration: 1-3 seconds each
- Volume: Normalized to -3dB

Once files are added, they play automatically during gameplay!
